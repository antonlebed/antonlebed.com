How It Works

Fair comparison: Both sides use the exact same algorithm. For each set of 5 primes, we generate all "smooth" numbers (products of prime powers) up to 100,000, with total exponent sum at most 12. For each of 24 measured SM parameters, we find the ratio a/b closest to the measured value. Same search, same limits, same scoring.

Why {2,3,5,7,11} wins: They are the first five primes, which maximizes smooth number density. But density alone gives ~0.05% mean error for ANY set of 5 small primes. The axiom's deeper advantage is structural: these primes define the ring Z/970200Z, and 24 physics constants happen to have clean expressions in this ring.
